# ai-vision-search-lab
Multimodalna wyszukiwarka semantyczna wykorzystująca AI Vision (GPT-4o/5) oraz bazę wektorową Qdrant. Projekt umożliwia inteligentne indeksowanie i przeszukiwanie archiwów zdjęć w języku naturalnym. Zawiera moduł diagnostyki tokenów reasoning dla modeli nowej generacji. Stack: Python, Streamlit.
